- #Unitree
- Github：https://github.com/unitreerobotics/unitree_sdk2
- ## 安装
- ```bash
  # clone
  git clone https://github.com/unitreerobotics/unitree_sdk2.git
  # build
  mkdir build
  cd build
  cmake .. -DCMAKE_INSTALL_PREFIX=/opt/unitree_robotics
  # build examples
  make
  # install
  sudo make install
  ```
